# Program Migration Plan for the Ecological Crisis Volume

The ecological-crisis manuscript contains long programmatic sections, formulas, mapping rules, and reproducibility logic. To keep the printed book readable, these materials should be migrated to this application directory.

## Keep in the book
- Core formula and parameter definitions.
- Short methodological explanations.
- Essential ecological-domain mapping tables.
- Case summaries and interpretive figures.
- A concise appendix pointing to this repository directory.

## Move to the repository
- Complete Python/Julia/R code blocks.
- R(t) trajectory scripts.
- Critical-slowing-down diagnostics.
- Remote-sensing and ecological-data cleaning pipelines.
- Food-web topology extraction routines.
- Parameter-record templates.
- Long reproducibility protocols.
