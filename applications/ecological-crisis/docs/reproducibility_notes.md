# Reproducibility Notes: Ecological Crisis

For every ecological case, record: case name, ecosystem type, location, period, six-parameter estimates, source notes, uncertainty ranges, formula version, R(t) trajectory construction method, early-warning diagnostics, code version or commit hash, and deviations from the shared ICI core implementation.

## Minimum metadata

```text
case_id
ecosystem_type
location
period_start
period_end
D, C, S, F, W, M
D_log_std, C_log_std, S_log_std, F_log_std, W_log_std, M_log_std
source_notes
formula_version = linear_FWM_v4
```
