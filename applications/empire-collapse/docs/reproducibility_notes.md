# Reproducibility Notes

Minimum reproducibility record for each case:

1. Case name and historical period.
2. Six parameter estimates: D, C, S, F, W, M.
3. Log-standard-error or uncertainty band for each parameter.
4. Source notes for each proxy variable.
5. Formula version.
6. Code version: release tag or commit hash.
7. Whether R(t) is absolute or normalized to a reference period.
8. Diagnostic outputs: R(t), warning band, breakpoint notes, residual checks if available.

Recommended release practice: tag a repository release for each book edition.
