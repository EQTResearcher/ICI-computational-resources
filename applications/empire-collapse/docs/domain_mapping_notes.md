# Domain Mapping Notes: Empire Collapse

This application maps the six ICI parameters to imperial historical systems.

| ICI parameter | Empire-domain interpretation | Typical proxy variables |
|---|---|---|
| D | Administrative and functional differentiation | administrative office types, social-functional modules, ethnic/cultural categories |
| C | Effective interaction population and resource base | registered population, fiscal revenue, mobilizable troops, grain reserves |
| S | Parallel throughput of orders, taxation, logistics, and trade | dispatch volume, trade throughput, tax cycles, grain transport rates |
| F | Closed governance feedback loops | censorate-report-response loops, military command loops, market price feedback, local-central correction loops |
| W | Normalized response/update frequency | inverse travel/response time from center to periphery, message turnaround frequency |
| M | Institutional memory depth | legal code depth, archives, administrative precedents, cultural-canonical persistence |

The main diagnostic target is R(t), because empire systems are non-autonomous and their FWM term is usually much smaller than the biological CA1 anchor.
