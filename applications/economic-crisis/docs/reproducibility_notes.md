# Reproducibility Notes: Economic Crisis

For every crisis case, record: case name and period; D, C, S, F, W, and M estimates; source notes; uncertainty ranges; formula version; R(t) trajectory construction method; structural-break or early-warning diagnostics; code version or commit hash; and any deviations from the shared ICI core implementation.

Minimum metadata:

```text
case_id
period_start
period_end
D, C, S, F, W, M
D_log_std, C_log_std, S_log_std, F_log_std, W_log_std, M_log_std
source_notes
formula_version = linear_FWM_v4
```
