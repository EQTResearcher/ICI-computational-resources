# Program Migration Plan for the AI Complexity Volume

The AI-complexity manuscript contains long programmatic sections, architecture-analysis routines, formulas, mapping rules, and reproducibility logic. To keep the printed book readable, these materials should be migrated to this application directory.

## Keep in the book

- Core formula and parameter definitions.
- Short methodological explanations.
- Essential AI-domain mapping tables.
- Architecture summaries and interpretive figures.
- A concise appendix pointing to this repository directory.

## Move to the repository

- Complete Python/Julia code blocks.
- Computation-graph parsing routines.
- ICI/R(t) scripts for AI architectures.
- Architecture-comparison data templates.
- Critical-threshold and overload diagnostics.
- Long reproducibility protocols.
- Versioned source notebooks and raw code appendices.
