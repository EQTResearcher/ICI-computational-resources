# Reproducibility Notes

Recommended workflow:

1. Record each case in `data/human_social_transition_cases_template.csv`.
2. Verify that all six parameters are positive.
3. Confirm that the linear FWM formula is used.
4. Compute ICI and R(t) through the shared core implementation.
5. Save outputs, figures, and logs with a version tag.
6. Record uncertainty propagation settings and random seeds.
