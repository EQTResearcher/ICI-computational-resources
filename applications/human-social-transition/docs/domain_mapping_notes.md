# Domain Mapping Notes: Human Social Transition

This document records how the six ICI parameters are interpreted in the human-social-transition volume.

| Parameter | Human-social-transition interpretation | Notes |
|---|---|---|
| D | Functional differentiation types | Institutional, occupational, technological, or organizational differentiation. |
| C | Effective interaction volume | Population, institutional nodes, transactions, or communicative connections. |
| S | Throughput | Rate of information, material, administrative, or economic processing. |
| F | Effective feedback loops | Regulatory, institutional, social, market, or information feedback structures. |
| W | Normalized interaction frequency | Response frequency relative to the ICI normalization convention. |
| M | Memory complexity | Archives, institutions, technical knowledge, cultural memory, and procedural memory. |

All estimates should include source, method, uncertainty, and version metadata.
